
# Прогулки по Пскову

Интерактивная карта прогулок по Пскову с возможностью выбора маршрутов, просмотра точек интереса и сохранения избранных маршрутов.

## Возможности

- Интерактивная карта с отображением маршрутов
- Фильтрация маршрутов по категориям (городские, природные, исторические, для начинающих)
- Поиск маршрутов по названию и описанию
- Сохранение избранных маршрутов
- Адаптивный дизайн для мобильных устройств
- Ленивая загрузка изображений для оптимизации производительности
- Кэширование данных для быстрой загрузки

## Установка и запуск

1. Установите Node.js, если он еще не установлен.

2. Клонируйте репозиторий или скачайте файлы проекта.

3. Установите зависимости:
   ```
   npm install
   ```

4. Подготовьте проект (рекомендуется):
   ```
   npm run prepare
   ```
   Эта команда последовательно выполнит:
   - Разделение маршрутов по категориям
   - Оптимизацию изображений
   - Сборку проекта

5. Или выполните команды по отдельности:
   - Разделение маршрутов по категориям:
     ```
     npm run process-routes
     ```
   - Оптимизация изображений:
     ```
     npm run optimize-images
     ```
   - Сборка проекта:
     ```
     npm run build
     ```

6. Откройте `index.html` в браузере для десктопной версии или `mobile.html` для мобильной версии.

## Оптимизация изображений

Для оптимизации изображений используется библиотека Sharp. Запустите скрипт оптимизации:

```
npm run optimize-images
```

Оптимизированные изображения будут сохранены в директорию `images_optimized`.

## Обработка маршрутов

Для разделения маршрутов по категориям используйте скрипт:

```
npm run process-routes
```

Это создаст отдельные файлы для каждой категории маршрутов в директории `data`.

## Сборка проекта

Для создания готовой к развертыванию версии проекта используйте команду:

```
npm run build
```

Эта команда создаст директорию `build` со всеми необходимыми файлами проекта, включая оптимизированные изображения (если они были созданы).
