
# Псковские тропы

Интерактивный путеводитель по пешеходным маршрутам в Пскове и его окрестностях.

## Структура проекта

```
Прогулки по пскову/
├── index.html              # Главный HTML-файл
├── css/
│   └── style.css           # Стили проекта
├── js/
│   ├── app.js              # Основной файл приложения
│   ├── map.js              # Функциональность карты
│   ├── routes.js           # Работа с маршрутами
│   └── ui.js               # Интерфейс и взаимодействие
├── data/
│   └── routes.json         # Данные о маршрутах
├── components/
│   ├── header.html         # Компонент шапки
│   ├── footer.html         # Компонент футера
│   └── route-modal.html    # Компонент модального окна маршрута
└── README.md               # Документация проекта
```

## Технологии

- HTML5
- CSS3 (с переменными и анимациями)
- JavaScript (ES6+)
- Leaflet.js (карты)
- Font Awesome (иконки)

## Разработка

Для запуска проекта достаточно открыть файл `index.html` в браузере. Для разработки рекомендуется использовать локальный веб-сервер.
