# Инструкция по добавлению фотографий

## Структура директорий для изображений

Для корректной работы приложения необходимо создать следующую структуру директорий:

```
/images/
├── routes/      // Фотографии маршрутов
└── points/      // Фотографии точек маршрутов
```

## Форматы изображений

- Используйте форматы JPG или PNG
- Оптимальный размер для фотографий маршрутов: 800x400 пикселей
- Оптимальный размер для фотографий точек: 600x400 пикселей
- Сжимайте изображения для быстрой загрузки (не более 200 КБ для маршрутов и 150 КБ для точек)

## Именование файлов

- Фотографии маршрутов: `[id-маршрута].jpg` (например, `1.jpg`, `2.jpg` и т.д.)
- Фотографии точек: `[id-маршрута]-[номер-точки].jpg` (например, `1-1.jpg`, `1-2.jpg` и т.д.)

## Пример

Для маршрута с id=1 и тремя точками:
- Фотография маршрута: `/images/routes/1.jpg`
- Фотографии точек: `/images/points/1-1.jpg`, `/images/points/1-2.jpg`, `/images/points/1-3.jpg`

## Обновление данных

Добавьте пути к фотографиям в файл `data/routes.json`:
```json
{
  "id": 1,
  "title": "Название маршрута",
  "image": "1.jpg",
  "points": [
    {
      "title": "Точка 1",
      "image": "1-1.jpg"
    },
    {
      "title": "Точка 2",
      "image": "1-2.jpg"
    }
  ]
}
```
